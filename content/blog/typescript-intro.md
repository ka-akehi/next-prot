TypeScript は JavaScript に型を加えたスーパーセットです。

## なぜ TypeScript か？

- IDE 補完が強力
- バグを事前に防止
- チーム開発に必須

## 型の基本

```ts
let count: number = 0;
let username: string = 'Taro';
let isActive: boolean = true;
```

## 関数の型

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

## まとめ

TypeScript はモダンなフロントエンド開発のスタンダードです。
